# Cal Poly Pomona's AVL Thermal Object Detection

**To clone this repository**: 
```bash
git clone https://github.com/Coolguy4123/CPP_AVL_ObjDet.git
```

## Steps to download the dataset
1. Go to https://adas-dataset-v2.flirconservator.com/#downloadguide and follow the instructions for installation
2. Once downloaded, move *"images_thermal_train"* and *"images_thermal_val"* into the working directory
